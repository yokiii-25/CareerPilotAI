# 🚀 CareerPilot AI

An AI-powered career platform that helps users improve their resumes, analyze ATS compatibility, match job descriptions, generate cover letters, and prepare for interviews.

---

## ✨ Features

- 📄 Resume Analyzer
- 📊 ATS Score Checker
- 🎯 Job Match Analyzer
- ✍️ Resume Improvement Suggestions
- 💌 AI Cover Letter Generator
- 🎤 Interview Preparation
- 📈 Career Dashboard
